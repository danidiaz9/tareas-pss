# tareas-pss
Programación de Software de Sistemas CC3301

### Tarea 1: Comprimir
Implementación de una función que comprime múltiples enteros sin signo en uno solo mediante concatenación de bits.

### Tarea 2: Reemplazo y Reemplazar
Implementación de funciones para reemplazar caracteres en strings, tanto creando un nuevo string como modificando el original.

### Tarea 3: Eliminar Rango
Función que elimina nodos en un rango específico de una lista enlazada ordenada.

### Tarea 4: Consultar Diccionario
Programa que busca una llave en un archivo de diccionario usando hashing cerrado.

### Tarea 5: Rot13
Implementación de un algoritmo de ordenamiento que trabaja con strings cifrados con ROT13.

### Tarea 6: Circuitos y Arquitectura
Diseño de circuitos digitales y problemas de arquitectura de computadores.

### Tarea 7: Compilar
Programa que recorre directorios buscando archivos .c que necesitan recompilación.
